# CPP 00

## 📚 Description

This project is the introduction to C++ at 42.  
It focuses on the basics of **Object-Oriented Programming (OOP)**, the differences between C and C++, and the use of the standard library in C++98.

The goal is to get familiar with:
- Classes and objects
- Member functions
- Streams (`std::cout`, `std::cin`)
- Basic encapsulation
- Header/source separation

All exercises must comply with the **C++98 standard**.

## 📌 Exercises Overview
ex00 — Megaphone
A program that converts input text to uppercase using std::string.

ex01 — PhoneBook
A simple contact manager using classes and arrays.

ex02 — Account
Implementation of a banking system with static attributes and logging behavior.
